How to display Javascript text messages inside of an HTML web page

This document shows 6 different methods of using Javascript to output text messages from inside of the HTML web page:-

  1. HEAD section, linking the (.html) web page document to an external (.js) file...displays an alert message box.
  2. HEAD section, use pair of script tags to display an alert message box.
  3. BODY section, use pair of script tags to write a mixture of 'plain text/HTML/CSS' directly into the (.html) web page document itself.
  4. BODY section, use pair of script tags to send text output to the web browser console.
  5. BODY section, use Javascript web page onLoad="" event to call/run an external file function to display an alert message box.
  6. BODY section, use an external JQuery file function call to both 'Show/Hide' a web page footer section.
